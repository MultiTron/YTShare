Key behaviors

  • Back arrow returns to the Friends screen
  • Messages polled every 3 seconds while this screen is open
  • Own messages (red bubbles) right-aligned, received (dark) left-aligned
  • Avatar + name in top bar matches the friend
  • Send button disabled when input is empty
  • Auto-scroll to bottom on new messages
  • If no chat exists yet, one is created on first message send
